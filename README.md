# 🌊 The Other Side

> 2025 경기 메타버스 해커톤 입상작  
> 도시 침수 재난 속, 보이지 않는 ‘시간 재난’의 발자취를 따라가는 인터랙티브 경험

---

## 🎮 프로젝트 개요

- **프로젝트 명** : The Other Side  
- **대회** : 2025 경기 메타버스 해커톤  
- **팀 명** : C.T.  
- **개발 기간** : 2025.08.18 ~ 2025.08.27  
- **개발 인원** : 기획 1명 / 개발 2명  
- **개발 도구** : Unity 6, Blender, GitHub Desktop, Rider  

도시가 기록적인 폭우로 침수 위기에 처한 상황에서,  
플레이어는 **자연물의 특성을 활용한 물리적 상호작용**으로 재난 현장을 헤쳐 나갑니다.  

플레이어가 지나온 동선과 행동은 사실,  
매년 수많은 ‘시간 재난’ 현장에서 **고귀한 희생을 감수한 구조대원들의 마지막 발자취**였다는 것이 드러나며,  
보이지 않는 곳에서 희생된 이들의 숭고한 노력을 마주하게 되는 **내러티브 기반 재난 체험형 게임**입니다.

## 👨‍💻 담당 업무

### 🗺 레벨 디자인 & 환경 구성
  - 언리얼 기반의 도시/자연 에셋을 Unity용으로 변환 및 레벨 디자인
  - 물 높이 변화에 따라 오브젝트가 자연스럽게 부유하는 시스템 구현  
  - 부력 방향 및 크기를 조절해, 떠다니는 나무/쓰레기/구조물 등이 연출과 게임플레이에 동시에 기여하도록 설계

### ✨ 쉐이더 & VFX
  - 폭우 연출 VFX
  - 흙탕물 표현 쉐이더
  - 파편 및 잔해 VFX

### 🧩 플레이어 미션 & 상호작용

#### 1. 장비 획득(줍기) 상호작용
  - 삽과 도끼를 줍는 상호작용 로직 구현  
  - 시야 내 상호작용 오브젝트 감지 
  - 상호작용 키 입력 시 장비 획득

#### 2. 실시간 터레인 상호작용 – 삽으로 땅 파기
  - 플레이어가 삽을 이용해 실시간으로 지형을 파내리는 시스템 구현
  - Unity Terrain 데이터를 런타임에 수정


#### 3. 콜라이더 상호작용 – 도끼로 나무 자르기
  - 도끼와 나무 간 콜라이더 상호작용을 이용해 나무 벌목 시스템 구현
